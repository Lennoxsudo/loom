设计原则

  • 一条时间线，一种行模板 — 不再是「term 一套 + 灰字一套」。
  • 固定四列网格 — gutter · verb · main · meta，扫读像表格。
  • 动词短、小写、等宽 — list read tree exec edit write。
  • 终端不是特例视觉 — 仍是同一行，只是 main 里带 $。
  • 思考 = 分隔符 — 无框文字；工具簇夹在两次思考之间，略紧凑。
  • 默认折叠 — 可展开的才有 chevron；纯 list/tree 用小点。
  • 右栏 meta 只放数字 — 行数 / 条目数 / exit · duration。
